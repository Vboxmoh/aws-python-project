# AWS Python Project

## Project Overview
This project demonstrates the fundamentals of Python programming executed on an AWS EC2 instance running Ubuntu 24.04 LTS.

## Environment
- AWS EC2 instance (Ubuntu 24.04 LTS)
- Python 3
- Connected via SSH

## Steps Completed

### 1. Syntax Practice
Printed name and age using the print() function.

### 2. Variables and Data Types
Created variables using different data types : string, integer and boolean.

### 3. Lists and Dictionaries
Created a list of favorite fruits and a dictionary with personal information key-value pairs.

### 4. Conditional Statements
Written a program that checks if a number is positive, negative or zero using if, elif and else statements.

### 5. Loops
Used a for loop to print each letter of a name and a while loop to count from 1 to 5.

### 6. Functions
Defined a reusable function that takes two numbers as parameters and returns their sum.

### 7. Modules
Created a custom module with two functions and imported it into a main file to reuse the code.

## Key Concepts Learned
- Python syntax and rules
- Variables and data types (string, int, float, boolean)
- Data structures (lists, dictionaries)
- Conditional statements (if, elif, else)
- Loops (for loop, while loop)
- Functions and parameters
- Modules and imports
